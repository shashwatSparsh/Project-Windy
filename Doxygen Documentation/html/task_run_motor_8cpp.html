void runMotor ( void *  p_params)

Task that runs Motor.

The task defines the motor input 1 and 2 pins as const 8bit ints based on the DRV8871 being used for motor control. The current angle is pulled from the queue and modified by a proportional gain to get a speed to turn the motor. Based on the sign of the angle, either input 1 or 2 are sent a PWM signal with the speed to rotate with the other set to LOW. The DRV8871 uses two inputs 1 and 2 as a way of directly controlling the direction of rotation. Determining whether to use input 1 or 2 based on the sign (+-) of the angle was done experimentally.
